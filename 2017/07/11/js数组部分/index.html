js基础知识 - 数组部分

发表于 2017-07-11 |

背景说明:最近重新看了遍 javascript 权威指南 数组部分，发现有一些原生的方法自己记得并不熟，这样不太好…

定义：
数组是值的有序集合，每个值叫做一个元素，而每个元素在数组中有一个位置，以数字表示称为索引。
数组是动态的，根据需要增长或缩减，数组可能是稀疏的，数组元素的索引不一定要连续，他们之间kennel有空缺
针对非稀疏数组，length属性就是数组元素的个数。针对稀疏数组 length属性 比元素个数要大
数组是对象的特殊形式

一、创建数组的方法
1.数组直接量 在方括号中将数组元素用逗号分隔开 eg: [1,2,3,4] (注： 数组直接量可以有选择的以逗号结尾 [4,4,] 两个元素)
2.调用构造函数Array( )创建数组
var a=new Array( ); // 创建一个没有数组参数
var a=new Array(10);
var a=new Array(10,e,w,d);

平时用使用数组字面量比较多因为更简单呀

二、数组元素的读和写
1.使用[ ]操作符来访问数组中的一个元素

  1. 数组是对象的特殊形式，使用方括号访问数组元素就像使用方括号访问对象的属性一样。js 将指定的数字索引值转换成字符串（索引值1 变成 “1” 然后将其作为属性名来使用）
    3.数组的特别之处在于，数组会自动维护其length 属性值。
    4.所有索引都是属性名，但只有在0～2的32次方 -2 之间的整数属性名才是索引。
    5.可以使用负数或非整数 来索引数组，这种情况下 数值转换为字符串，字符串作为属性名来用。
    既然、不是非负整数，它就只能当常规的对象属性，而非数组的引用
    6.任何不存在的属性 都是 undefined
    7.既然数组是对象，那么它们可以从原型中继承元素

三、稀疏数组
1.稀疏数组就是 包含从0 开始的不连续索引的数组
2.足够稀疏的数组通常在实现上比稠密的数组更慢、内存利用率更高，在这样的数组中查找元素
的时间与常规对象属性的查找时间一样长
3.数组直接量中省略值时不会创建稀疏数组，省略的元素在数组中是存在的，其值为 undefine
这和数组元素根本不存在是有微妙的区别的，可以用in操作符检测两者之间的区别
eg:
var a1=[ , , ,]; //数组是 [undefined,undefined,undefined]
var a2=new Array(3); //该数组根本没有元素
0 in a1 //true ：a1在索引0处有一个元素
0 in a2 // false : a2在索引0处没有元素

四、数组长度
1.每个数组有一个 length属性，这个属性使其区别于常规的对象。
2.非稀疏数组，length值代表数组中的元素个数。其值笔数组中最大的索引大1
3.当数组是稀疏时，length属性的值大于元素的个数
4.数组的两个特殊行为：1.如果为一个数组元素赋值，它的索引i大于或等于现有数组的长度时
length属性的值将设置为i+1 。2.设置length 属性值为大于其当前的长度。 实际上这不会向数组中国年天津
新的元素，它只是在数组尾部创建一个空的区域。
a =[1,2,3,4,5];
a.length=3; //改变长度后 现在a=[1,2,3]
a.length=0; //删除所有元素 a=[ ]
a.length=5;// 长度为5，但是没有元素 就像 new Array(5)
5.ES5中 Object.defineProperty( ) 让数组变成只读
eg: a=[1,2,3]
Object.defineProperty(a,”length”,{ writable:false } );
a.length=0 //a 不会改变的
五、数组方法
1.为新索引赋值：
a=[ ] ; a[0]=“zero”; a[1]=“one”;
2.push( ) 和 pop( )
push( ) 方法在数组末尾添加一个或多个元素,并返回数组新的长度。
pop( )方法是删除数组中的最后一个元素，并返回删除的值
var stack=[ ];
stack.push(1,2); //返回数组长度2 stack =[1,2]
stack.pop(); //返回删除元素 2 stack=[1]
stack.push([4,5]); // 返回数组长度2 stack=[1,[4,5]]
stack.pop() //返回被删除元素 [4,5] stack=[1]
3.unshift( ) 和 shift( )
unshift( ) 在数组的头部插入一个元素 或多个元素 。返回数组的长度
shift( ) 删除数组的第一个元素并返回删除元素。
var a=[ ];
a.unshift(1); //返回 1 当前a=[1]
a.unshift(22) //返回 2 当前a=[22,1]
a.shift() //返回 22 当前a=[1]
a.unshift(3,[4,5]) //返回 3 当前a=[1,3,[4,5]]
unshift() 参数是一次性插入的
4.delete
var a=[1,2,3];
delete a[1] //a在索引1的位置不再有元素
a.length // 3 delete 操作并不影响数组的长度
*删除数组元素 数组长度不变 ，变成稀疏数组

5.join( )
Array.join( ) 方法将数组中的所有元素都转化为字符串并拼接在一起，返回最后生成的
字符串。可以指定间隔符号，不指定默认为逗号
var a=[1,2,3];
a.join(); //返回 “1，2，3”
a.join(“.”) //返回 “1.2.3”
6.reverse( )
Array.reverse( )方法将数组中的元素顺序颠倒 并返回逆向的数组。
7.sort( )
Array.sort( ) 方法将数组中的元素排序并返回排序后的数组
当不带参数调用sort( )时，数组元素以字母表顺序排序排序，如果有undefined 元素会被拍到最后（如有必要将临时转化为字符串比较）
var a=[33,4,5,6]
a.sort(); // 字母顺序排序：33 4 5 6
a.sort(function(a,b){ //数值 顺序 4，33，222，1111
return a-b; //根据顺序 负 0 正
});
a.sort(function(a,b){ //数值大小相反的顺序
return b-a;
});
8.concat( )
Array.concat( ) 方法创建并返回一个新数组 ,它的元素包括调用concat( ) 的原始数组元素和concat( )的每一个参数，如果这些参数中的任何一个自身是数组，则链接的是数组元素，而非数组本身。但concat( )不会递归扁平化数组的数组
var a=[1,2,3];
a.concat(4,5) //返回 [1,2,3,4,5]
a.concat([4,5]) //返回 [1,2,3,4,5]
a.concat(4,[5,[6,7]]) ////返回 [1,2,3,4,5,[6,7]] concat( )不会递归扁平化数组的数组
9.slice( )
返回指定数组的一个片段和子数组，它的两个参数分别指定了片段的开始和结束
的位置。包含开始位置元素 不包含结尾位置元素,如果只指定了一个参数，返回的数组将包含从开始位置到数组 结尾的所有元素。如果参数出现负数 则指倒数 如：-1 倒数第一个 ，-2倒数第二个。
var a=[1,2,3,4,5];
a.slice(0,3) //[1,2,3]
a.slice(3) //返回[4,5]
10.splice( )
在数组中插入或删除元素的通用方法。不同于slice( ),concat( ),splice( ) 会修改调用的数组 。
splice( ) 能够从数组中删除元素、插入元素到数组中 或者同时完成两种操作
.第一个参数指定插入／删除的起始 位置
.第二个参数指定了从数组中删除元素的个数，如果省略第二个参数，起始点开始到数组结尾的所有元素都将被删除。
结果返回 被删除元素的数组
var a=[1,2,3,4,5,6,7,8];
a.splice(4); 返回 [5,6,7,8] 当前a [1,2,3,4]
a.splice(1,2);返回[2,3] 当前a [1,4]
前两个参数指定了需要删除的数组元素，紧随其后的任意个数的参数指定了要插入到数组中的元素，从第一个参数指定的位置到开始插入
var a=[1,2,3,4,5]
a.splice(2,0,’a’,’b’);// 从第二个位置开始，删除0个 并插入’a’,’b’ 返回[ ]当前a为[1,2,’a’,’b’,3,4,5 ]
a.splice(2,2,[1,2],3); //返回[’a’,’b’] a是[1,2,[1,2],3,3,4,5]
区别于concat( ) ,splice( ) 会插入数组本身而非数组的元素
11.toString( ) 和 toLocaleString( )
toString( ) 方法将每一个元素转化为字符数，并且输出用逗号分隔的字符串列表，注意输出不包括 方括号火灾其她人和形式的包裹数组值的分隔符号
[1,2,3].toString() //‘1,2,3’
*同 [1,2,3].join() 一样
toLocaleString( )是toString( )方法的本地化版本，它调用元素的toLocaleString( )方法将每个数组元素转化为字符串，并且实用本地化分隔符将这些字符串链接起来形成最终的字符串
12.数组类型
给定一个未知的对象，判定是否为数组 ES5中可以使用Array.isArray( )函数判定
Array.isArray([]) //true
Array.isArray({}) //false
typeof操作符，对数组它返回“对象”
instance 操作符 只能用于简单的情形：
[ ] instanceof Array //=>true
({ }) instanceof Array //=>false
使用instanceof的问题是在web浏览器中有可能有多个窗口或窗体（frame），每个窗体都有自己的javascript环境，有自己的全局变量。并且，每个全局对象有自己的一组构造函数。因此一个窗体中的对象将不可能是另外窗体中的构造函数的实例。窗体之间的混淆不常发生，但是这个问题已证明 instanceof 操作符不能视为一个可靠的数组检查方法。

13.indexOf( ) 和 lastIndexOf( )
indexOf( ) 和 lastIndexOf( ) 搜索整个数组中具有给定值的元素，返回找到的第一个元素索引 或者如果没有找到就返回 -1 。indexOf( ) 从头至尾搜索，而lastIndexOf( ) 则返向搜索。
一共两个参数，第一个是需要搜索的值，第二个可选参数，指定数组中的索引，从哪里开始搜索，没有的话 indexOf( ) 从第一个开始， lastIndexOf( ) 从最后一个开始
六、数组遍历、映射、过滤、检测、简化和搜索
ECMAScript5中的数组方法简述：
首先大多数方法的第一个参数接收的是一个函数，并对数组的每一个元素调用一次该函数。如果是稀疏数组，对不存在的元素不调用传递的函数。在大多数情况下，调用提供的函数使用三个参数，数组元素、数组的索引和数组本身。大多数数组方法的第一个参数是一个函数，第二个参数是可选的。如果有第二个参数，则调用函数被看做是第二个参数的方法，也就是说，在调用函数时传递的第二个参数作为它的this 关键字的值来使用 。ECMAScript5中的 数组方法都不会修改它们调用的原始数组，当然传递给这些方法的函数可以修改

1.forEach( )
方法从头至尾遍历数组，为每个元素调用指定的函数

//求数组的和
var data=[1,2,3,4,5];
var sum=0;
data.forEach(function(value){sum + =value;})
//数组的每个元素加1
data.forEach(function(v,i,a){ a[i]=v+1; }); //data =[2,3,4,5,6]

注意： forEach( )无法在所有元素都传递给调用的函数之前终止便利，没有类似 for循环中的break 。如果要提前终止，必须把 forEach( )方法防盗try 块中，并能抛出一个异常。如果 forEach( ) 调用的函数抛出 foreach.break 异常，循环提前终止

2.map( )
方法将调用的数组的每一个元素传递给指定的函数，并返回一个数组，包含该函数的返回值。

a=[1,2,3];
b=a.map(function(x){return x*x})

//b=[1,4,9] map返回的是新数组，不修改调用的数组。与forEach不同,forEach会改变原数组

3.filter( )
方法返回的元素是调用数组的一个子集，传递的函数是用来逻辑判定的：该函数返回 true 或 false 调用判定函数就像调用forEach map一样。如果返回true ，那么传递给判定函数的元素就是这个子集的成员，它就被添加到一个作为返回值的数组中。

a=[5,4,3,2,1]
a.filter(function(x){return x<3}) //返回[2,1]
//filter会跳过稀疏数组中缺少的元素，它返回数组中是稠密的
eg:
a=a.filter(function(){return true;})
//压缩空缺并删除undefined、null
a=a.filter(function(x){return x!==undefined && !==null})

4.every( ) 和 some( )
every( ) 和 some( )方法是数组的逻辑判定：它们对数组元素应用指定函数进行判定，返回true or false
every( ) 数组中所有元素都符合 返回true 有一个或多个不符合 返回false
some( ) 数组中元素有符合的返回 true ，全都不符合 返回false

5.reduce( ) 和 reduceRight( )
reduce( ) 和 reduceRight( ) 方法使用指定的函数将数组元素进行组合，生成单个值。这在函数式中是常见的操作，也可以称为“注入” 和“折叠” 。举例说明它是如何工作的：
var a=[1,2,3,4,5]
var sum=a.reduce(function(x,y){ return x+y },0);// 数组求和
reduce( )有两个参数，第一个是执行化简操作的函数。化简函数的任务就是用某种方法把两个值组合 或化简为一个值，并返回化简后的值。第二个（可选）参数是一个传递给函数的初始值。
reduce( ) 使用的函数与forEach( )、map( ) 使用的函数不同。数组元素、索引、数组本身 作为 2～4 个参数传递给函数。第一个参数是目前为止的化简操作累积
